# ステラブリーダー (Stella Breeder) 🌌🏇

星たちの導きと血統のケミストリーが織りなす、本格派・競走馬育成配合シミュレーションゲーム。  
あなただけの牧場を開き、数多の名牝馬と伝説の種牡馬たちを掛け合わせ、完璧なインブリード・ニックスバランスを追い求め、ターフの絶対王者を目指しましょう。

---

## 🌟 主なゲーム特徴

### 🧬 本格的な配合理論・血統システム
- **インブリード（クロス）**: 祖先5代前までのペディグリー（血統表）を解析。名前の共通文字から「因子のクロス」を自動検知し、スピード、スタミナ、勝負根性などのパラメータを限界突破させます。
- **ニックス相性**: 異なる系統や名前の響きの交差によるケミストリー（相乗効果）を計算。完璧なニックス相性を見つけ出すことで、本来の能力以上の爆発力を引き出します。
- **4大系統（LINEAGES）**:
  - ⚡ **スピード系**（例: アルタイルダッシュ、流星王、シリウスオーブ） — 爆発的な速さ、スプリント戦の絶対王者
  - 🔋 **スタミナ系**（例: ベテルギウスキング、カノープススピリット） — 無尽蔵の体力、ステイヤーズG1を制する底力
  - 🔥 **根性系**（例: リゲルブレイブ、アンタレスフレイム、神龍帝） — ゴール前の競り合いを絶対に落とさない不屈の闘志
  - ☯️ **バランス系**（例: デネブロード、スピカハート、ポルックスブルー） — どんな馬場や距離にも順応する万能性

### 🧠 Gemini搭載・専属配合AIアドバイザー
- 配合に悩んだ時は「おすすめ」や質問、対話でアドバイスを取得。
- 選択中の繁殖牝馬に完璧に適合する一押しの「配合パートナー」のレコメンドや、血統クロスの持つポテンシャル、将来のG1栄冠への見通しを、世界観に合わせた表現で丁寧に解説してくれます。

### 🏎️ リアルなレースシミュレーター＆成長システム
- 誕生した幼駒の「成長型」（早熟・普通・晩成）と脚質（逃げ・先行・差し・追込）を考慮。
- スピード、スタミナ、根性、気性、健康、幸運などの詳細なステータスと調教ログ。
- レース展開をリアルに可視化するドット風/グラフィカルなインジケーターで、手に汗握るG1レースを心ゆくまで体験できます。

## 🛠️ 技術スタックと構成

- **フロントエンド**: React 18, Vite 6, Tailwind CSS (v4)
- **アニメーション**: Motion (Framer Motion)
- **アイコン**: Lucide React
- **API連携**: `@google/genai` (Gemini APIの最新サーバーサイドプロキシ連携)
- **Webサーバー**: Node.js, Express (フルスタック設計。API Keyなど機密性の高い情報は常にサーバーサイドで安全に管理・疎通されます)

## 🚦 ローカルでの起動方法

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで、以下を実行してライブラリをインストールします。
```bash
npm install
```

### 2. 開発サーバーの起動（ポート3000）
ローカル開発用のサーバーを起動します。
```bash
npm run dev
```
起動完了後、 [http://localhost:3000](http://localhost:3000) にアクセスします。

### 3. アプリケーションのビルド
本番運用（ホスティングやPagesデプロイ等）のためにソースコードをビルドする場合：
```bash
npm run build
```
ビルド成果物は `dist/` ディレクトリに生成されます。

### 4. 本番用カスタムサーバーのローカル実行
```bash
npm start
```

## 🔒 セキュリティとAPIキーについて

本プロジェクトは **APIキーや認証情報をブラウザ側（クライアントサイド）に一切ハードコードしない、セキュアなフルスタック構成** で作られています。
- ユーザーにAPIキーの入力を求める不要なUIは排除されています。
- 外部APIへの通信は、Expressサーバー上の安全なバックエンドAPI(`/api/*` などのプロキシ)を通じて行われるため、秘密鍵やシステムアクセス権が第三者に公開される心配はありません。
- noteやSNSへの安全なゲーム画面の共有、GitHubへの安心なレポジトリ公開をそのまま行っていただけます。
